___
## First - Introduction

- Rails 7.0.0
- Ruby 3.2.1

- Docker
    - docker-compose up -d

    - Run postgres on port 5450
    - Run redis on port 6379

- Install the dependencies and create the database

    ```ruby
    bin/setup
    ```

- Run the rails server

    ```ruby
    bin/dev
    ```
    based on the file **Procfile.dev**
